# Card-Tradder – Frontend

Este directorio contiene el frontend estático de la aplicación:

- `public/index.html` → página principal
- `public/styles.css` → estilos
- Imágenes e íconos dentro de `public/`

El backend (en `../backend/server.js`) sirve esta carpeta directamente como archivos estáticos, por lo que no se requiere build ni framework adicional.

Para trabajar en el frontend:

1. Edita `public/index.html` y `public/styles.css` según sea necesario.
2. Inicia el backend desde `../backend` con `npm start` y abre `http://localhost:3000`.
