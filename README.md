# CompraVenta_Pokemon
compra venta de cartas pokemon con subasta

## Cómo subir cambios usando GitHub Desktop

1. Abre **GitHub Desktop** y selecciona el repositorio local: `File > Open` y apunta a la carpeta `CompraVenta_Pokemon`.
2. Asegúrate de estar en la rama correcta (por ejemplo, `develop` o `main`) con el selector de ramas en la parte superior.
3. Verifica los archivos modificados en la pestaña **Changes**; si falta alguno, usa "Show in Finder/Explorer" para confirmar la ruta.
4. Escribe un mensaje de commit claro en **Summary** y haz clic en **Commit to <rama>**.
5. Pulsa **Push origin** (o **Publish repository** si es el primer push) para enviar los cambios a GitHub.
6. Si necesitas un Pull Request, abre **View on GitHub** desde GitHub Desktop y crea el PR desde la rama actual hacia la rama destino.
